BMI APP

این پروژه یک برنامه گرافیکی محاسبه BMI با استفاده از Java Swing می‌باشد.

Workshop11

 امکانات برنامه
- صفحه خوش‌آمدگویی
- محاسبه BMI بر اساس قد و وزن
- نمایش وضعیت بدنی (کم‌وزن، نرمال، اضافه‌وزن، چاق)
- ارزیابی بر اساس جنسیت
- امکان تغییر رنگ پس‌زمینه
- رابط کاربری ساده و کاربرپسند

 صفحات برنامه
 
 صفحه اول
نمایش توضیحات کلی برنامه و ورود به صفحه محاسبه BMI

 صفحه دوم
دریافت قد و وزن کاربر و نمایش:
- مقدار BMI
- وضعیت بدنی
